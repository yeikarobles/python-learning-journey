Comparison Operators

| OPARATOR | MEANING                  |
| -------- | ------------------------ |
| >        | Greater than             |
| <        | Less than                |
| >=       | Greater than or equal to |
| <=       | Less than or equal to    |
| ==       | Equal to                 |
| !=       | Not equal to             |
| %        | Return remain of "/" op. |

MODULO OPARATOR (%)
return 0 if is an even number because there would be no remaining left of the division / 2
